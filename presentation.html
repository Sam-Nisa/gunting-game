Metal Fury Gunting

2D Action Survival Web Game

Built using JavaScript & HTML5 Canvas

What is This Game?

  • Fast-paced survival shooting game.
  • Fight endless waves of enemies.
  • Playable on PC and Mobile.

Core Gameplay

Every match follows a tight, addictive loop designed to keep players engaged and always pushing for the next level.

Game Engine

  • 60 FPS smooth gameplay using requestAnimationFrame.
  • Custom physics and collision system.
  • Difficulty increases automatically.

Enemies & Bosses

  • Different enemy types with unique behavior.
  • Flying drones using Math.sin movement.
  • Boss appears every 5 levels.

Audio System

  • No MP3 files used.
  • Sounds generated using Web Audio API.
  • Lightweight and fast loading.

Backend (Firebase)

  • Anonymous login system.
  • Save scores and coins online.
  • Global leaderboard system.
